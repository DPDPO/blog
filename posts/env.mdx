---

ignore에 .env 파일을 입력하여 원격저장소에 노출되지 않게 할려했는데 이미 원격저장소에 .env파일이 있어서 gitignore 파일에 써놔도 의미가 없었다.

<br />
<br />
원격 저장소의 env파일을 지워 해결했다.

<br />
<br />
git rm .env --cached git commit -m "remove .env file from git repo" git push
origin main
