**lodash?**

보통의 경우 array, collection, date 등 데이터의 필수적인 구조를 쉽게 다룰 수 있게 끔 하는데에 사용<br />
JavaScript에서 배열 안의 객체들의 값을 handling(배열, 객체 및 문자열 반복 / 복합적인 함수 생성) 할 때 유용하다.<br />
ㅡ. (변수) 이런식으로 작성할 경우 lodash wrapper로 변수를 감싸게 되면서 해당 변수에 대한 chaining을 시작한다.

<br />
**array 관련 method**<br />
(1)findIndex()<br />
형식: _.findindex(array,[predicate=.indentity],[thisArg])<br />
출력: index number<br />
배열 내에서 원하는 index를 쉽게 구할 수 있다.
<br />
```tsx
ex)
  var users = [
    { 'user': 'barney',  'active': false },
    { 'user': 'fred',    'active': false },
    { 'user': 'pebbles', 'active': true }
  ];
 
  _.findIndex(users, function(o) { return o.user == 'barney'; });
  // => 0
 
  // The `_.matches` iteratee shorthand.
  _.findIndex(users, { 'user': 'fred', 'active': false });
  // => 1
 
  // The `_.matchesProperty` iteratee shorthand.
  _.findIndex(users, ['active', false]);
  // => 0
 
  // The `_.property` iteratee shorthand.
  _.findIndex(users, 'active');
  // => 2
 /
```
<br />
(2)flatten()<br />
형식: _.flatten(arraym[isDeep])<br />
다차원 배열 내의 요소를 출력하는데 편리하다.
<br />
```tsx
ex)
_.flatten([1, [2, 3, [4]]]);
// → [1, 2, 3, [4]]

// 배열안의 배열 값을 깊이와 상관없이 순서대로 나열.
_.flatten([1, [2, 3, [4]]], true);
// → [1, 2, 3, 4]

```
<br />
(3)remove()<br />
형식: .remove(array, [predicate=.identity], [thisArg])<br />
출력: 제거된 array<br />
배열 내의 조건에 맞는 요소들을 제거한 후 반환해준다.
<br />
```tsx
ex)
var array=[1,2,3,4];

var evens=remove(array,function(n){
   return n%2==0;
});

console.log(array);
//-> [1,3]

console.log(evens);
//-> [2,4]

```

**collection 관련 method**<br />
(1)reduce()<br />
형식: .reduce(collection, [iteratee=.identity], [accumulator], [thisArg])<br />
<br />
```tsx
ex)
//첫번째 인자에 대해 배열 내부의 값을 통해 콜백함수를 실행시킨 후 결과값을 반환.

_.reduce([1, 2], function(total, n) {
  return total + n;
});
// → 3

```
<br />
(2)forEach()<br />
형식: .forEach(collection, [iteratee=.identity], [thisArg])<br />
배열의 값마다 함수를 실행시킬 때 용이하게 사용.
<br />
```tsx
ex)
_([1, 2]).forEach(function(n) {
  console.log(n);
}).value();
// 1
// 2
```
<br />
(3)every()<br />
형식: .every(collection, [predicate=.identity], [thisArg])<br />
출력: boolean 값<br />
배열 안 요소들의 값들을 비교하고 분석하는데 용이하다.
<br />
```tsx
ex)
var myFriend = [
  { name: 'kim', active: false },
  { name: 'lee', active: false }
];

// 값을 비교할 수 있다.
_.every(myFriend, { name: 'kim', active: false });
// → true

// key와 value가 있는지 확인할 수 있다.
_.every(myFriend, 'active', false);
// → true

// key에 해당하는 value가 모두 true이면 true를 반환.
_.every(myFriend, 'active');
// → false

```
